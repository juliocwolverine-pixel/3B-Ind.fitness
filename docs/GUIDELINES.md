# Diretrizes de Assets - 3B Indústria & Fitness

## 📐 Especificações de Imagens

### Logo
- **Formato**: PNG com fundo transparente ou SVG
- **Tamanho mínimo**: 200x50px
- **Resolução**: 72 DPI para web

### Banners Hero
- **Desktop**: 1920x600px (16:10)
- **Tablet**: 1024x400px (16:10)
- **Mobile**: 375x300px (1.25:1)
- **Formato**: JPG otimizado ou WebP
- **Peso máximo**: 500KB

### Ícones
- **Formato**: SVG (preferido) ou PNG
- **Tamanho**: 16x16, 32x32, 64x64px
- **Fundo**: Transparente

### Imagens de Produto
- **Tamanho mínimo**: 400x400px
- **Tamanho recomendado**: 800x800px
- **Formato**: JPG ou WebP
- **Peso**: ~200-400KB

## 🎨 Paleta de Cores

- **Primária**: #1A1A1A (Preto)
- **Secundária**: #FF6B35 (Laranja)
- **Terciária**: #004E89 (Azul)
- **Neutro**: #F5F5F5 (Branco off)

## 📏 Tipografia

- **Heading**: Montserrat Bold
- **Corpo**: Roboto Regular
- **Peso**: 400, 600, 700

## ✅ Checklist Antes de Upload

- [ ] Imagem está otimizada
- [ ] Nomeação segue padrão
- [ ] Resolução está correta
- [ ] Arquivo está em pasta correta
- [ ] Tamanho é adequado para web

## 🚀 Ferramentas Recomendadas

- **Otimização**: TinyPNG, ImageOptim
- **Design**: Figma, Adobe XD
- **Conversão**: CloudConvert (para WebP)
