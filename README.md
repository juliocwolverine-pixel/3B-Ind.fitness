## ⚡ Performance

Para melhor performance e cache:
- Use URLs via **Jsdelivr CDN** (recomendado)
- Otimize imagens antes de upload
- Use formatos modernos (WebP quando possível)

## 📋 Padrões de Nomeação

- Use nomes descritivos em minúsculas
- Separe palavras com hífen: `logo-completo.png`
- Inclua dimensões quando relevante: `hero-1920x600.png`
- Evite espaços e caracteres especiais

## 📦 Tipos de Arquivo Suportados

- **Imagens**: PNG, JPG, WebP, GIF, SVG
- **Ícones**: SVG (recomendado), PNG
- **Documentos**: PDF

## 🔄 Fluxo de Atualização

1. Clone o repositório
2. Adicione/modifique imagens nas respectivas pastas
3. Faça commit com mensagem descritiva
4. Push para a branch `main`
5. As mudanças serão refletidas em até 5 minutos (CDN pode cachear mais)
